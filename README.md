# DBM - Project

Un projet collaboratif pour créer une fusée artisanale propulsée au KNO₃ + sucre.

## Modules :

- Design moteur avec FreeCAD
- Propergol recrystalisé

## Dossier important

- `docs/` : designs CAO, schémas, versions précédentes
